# Backend Development Setup

## Prerequisites

- Java 21
- Gradle (wrapper included in project)
- IDE: IntelliJ IDEA (recommended)

## Setup

Clone the repository and navigate to:

```bash
cd sprintstart-backend
```

## Run the Application

```bash
./gradlew bootRun
```

Or build and run:

```bash
./gradlew build
java -jar build/libs/*.jar
```
